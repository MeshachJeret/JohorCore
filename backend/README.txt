Project Name: JohorCore

Aims to solve: Long wait times at the Johor-Singapore border and heavy congestion

The Solution: Real-time dashboard with traffic, weather, and AI assistance.

Tech Stack: Node.js, Express, Google Cloud Run, Gemini AI.

Live Link: https://johor-core-backend-526658808469.asia-southeast1.run.app/

If you would like to run this locally, please do run commands "npm install" and then type in "node server.js" to run the web app locally (input commands without the quotes)




NOTE: Please do hit Ctrl + F5 to hard reset the web app, therefore the current data from the api can be displayed
